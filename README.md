# Ethical Hacking Practical Assignments

## Overview

This repository contains practical ethical hacking and network security exercises completed as part of the BIT4139 Ethical Hacking course. The activities focus on reconnaissance, footprinting, scanning, enumeration, and vulnerability assessment using industry-standard cybersecurity tools.

## Objectives

* Install and configure penetration testing tools.
* Perform network reconnaissance and footprinting.
* Discover active hosts within a network.
* Conduct port scanning and service detection.
* Perform DNS footprinting.
* Identify potential security vulnerabilities.
* Analyze and document findings.

## Tools Used

* Kali Linux
* Nmap
* Oracle VirtualBox
* Windows Command Prompt
* DNS Analysis Tools

## Activities Performed

### 1. Environment Setup

* Installed Nmap.
* Configured Oracle VirtualBox.
* Deployed Kali Linux virtual machine.

### 2. Network Footprinting

* Identified network configuration and target devices.
* Discovered active hosts on the local network.

### 3. Port Scanning

* Scanned target systems for open, closed, and filtered ports.

### 4. Service Detection

* Identified running services and versions on open ports.

### 5. DNS Footprinting

* Gathered DNS-related information about target systems.

### 6. Vulnerability Assessment

* Identified security misconfigurations and potential weaknesses.

## Learning Outcomes

Through these exercises, I gained practical experience in:

* Network reconnaissance
* Enumeration techniques
* Vulnerability assessment
* Security analysis and reporting
* Using cybersecurity tools in a controlled and authorized environment

## Disclaimer

All activities were conducted in authorized environments for educational purposes only. This repository does not promote unauthorized access, exploitation, or malicious activities.
